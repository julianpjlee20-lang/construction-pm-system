### 方法一：使用啟動腳本（推薦）

```bash
cd notes/向上建設/工程專案管理系統/src/frontend

```

```bash
cd notes/向上建設/工程專案管理系統/src/frontend

npm run dev
```

## 📋 功能測試清單

啟動後請測試以下功能：

### ✅ Kanban 看板
1. 查看三欄式看板（待辦 / 進行中 / 已完成）
2. **拖拉「鋼筋綁紮」卡片**到「已完成」欄
3. 確認卡片移動成功
4. 點擊任意卡片開啟詳情頁

### ✅ 任務詳情頁
1. 修改任務名稱
2. 調整進度滑桿（例如：60% → 80%）
3. 修改負責人
4. 點擊「儲存」
5. 確認 Kanban 卡片更新

### ✅ 甘特圖
1. 切換到「甘特圖」分頁
2. 查看三個任務的時間軸
3. 確認顏色警示：
   - 「基礎開挖」：綠色（已完成）
   - 「鋼筋綁紮」：黃色或藍色（視進度而定）
   - 「模板組立」：藍色（尚未開始）
4. 點擊甘特圖任務條 → 開啟詳情頁

### ✅ 照片上傳
1. 在任務詳情頁中
2. 拖拉圖片到上傳區域（或點擊選擇）
3. 觀察上傳進度條（0% → 100%）
4. 確認照片出現在時間軸
5. 點擊「查看原圖」按鈕

## 🐛 常見問題

### Q: Port 3000 被佔用？
**A**: Vite 會自動使用下一個可用 port（3001、3002...），檢查終端訊息即可。

### Q: 畫面空白？
**A**: 
1. 檢查瀏覽器 Console（F12）是否有錯誤
2. 確認 `npm install` 是否成功
3. 嘗試 `npm run build` 檢查編譯錯誤

### Q: 拖拉功能不順暢？
**A**: 確認使用現代瀏覽器（Chrome、Firefox、Safari），不支援 IE。

### Q: 照片上傳後看不到？
**A**: 目前是 Mock 上傳（模擬），照片會顯示縮圖，點擊「查看原圖」會開啟測試連結。
